NOTES
========================================================================
- Backend: 5 boosters (TIME_FREEZE, HINT, SKIP, DOUBLE_POINTS, UNDO)
- Backend: 10 levels with scaling difficulty (groups 3→6, items 3→6)
- Backend: Combo engine (max 4× multiplier, milestone every 3)
- Backend: Daily challenge (SHA-256 seeded, bonus multiplier)
- Backend: Leaderboard (top 100) + score history
- State: Custom hooks (useState + useCallback), NOT Zustand
- No React Router — single GamePage with phase state machine
- Audio: 7 sounds via AudioManager pattern
- Word bank: 8 predefined sets, seeded selection
- Default config: 5s memorize, 60s play, 3 groups × 3 words
- Scoring: 50/group + 10/partial word + (sec × 3) time bonus
- Star rating: 3★ = all correct, 2★ = ≥66% correct, 1★ = completed
