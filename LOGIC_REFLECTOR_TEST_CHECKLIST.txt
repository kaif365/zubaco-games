 Star Rating System:
   ★★★ (3 stars) — Solved using minimum (optimal) number of block placements
   ★★☆ (2 stars) — Solved using optimal + 1 block placements
   ★☆☆ (1 star)  — Solved (any number of placements)

 Achievement Triggers:
   FIRST_WIN      — Win 1 game
   STREAK_5       — 5-level streak (consecutive solves)
   STREAK_10      — 10-level streak
   PERFECT_GAME   — Solve with minimum blocks (3★)
   SPEED_DEMON    — Finish with >50% time remaining
   PERSISTENCE    — Play 50 games
   LEVEL_5        — Reach level 5
   LEVEL_10       — Reach level 10
   HIGH_SCORER    — Score 1000+ in session
   NO_MISTAKES    — Complete without removing any block

 Block Types:
   reflect-block  — Square reflective surface (bounces beam off nearest face)
   mirror-fwd /   — Diagonal mirror (redirects beam 90°)
   mirror-bwd \   — Diagonal mirror (redirects beam 90° opposite)
   splitter +     — Splits beam into multiple directions
   blocker ■      — Absorbs/stops the beam
