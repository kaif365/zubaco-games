GAME CONCEPT
============================================================

"Block Fill" is a Flow Free clone. The player connects pairs of colored dots
on a grid by drawing paths (pipes) between them. Rules:
  1. Each color has exactly 2 endpoints (dots) on the grid
  2. Draw a path connecting both dots of the same color
  3. Paths cannot cross or overlap
  4. The ENTIRE grid must be filled (no empty cells)
  5. Success = all pairs connected + 100% board coverage

Difficulty scales by:
  - Grid size (5×5 easy → 9×9+ hard)
  - Number of color pairs (3 easy → 10+ hard)
  - Path complexity (longer winding paths = harder)

CURRENT ARCHITECTURE
============================================================

Tech Stack:
  - Frontend: React 19 + Vite + TypeScript + TailwindCSS 4 + Framer Motion
  - Backend: NestJS + Prisma + Bun + Restate SDK
  - UI Library: shadcn/ui components (Button, Card, Dialog)
  - Audio: Howler.js with custom AudioManager
  - Rendering: Both HTML/CSS (FlowBoard.tsx) AND Phaser 4 (PhaserFlowBoard.tsx)

WHAT TO MAKE CONSISTENT ACROSS ALL GAMES
============================================================

These features should work IDENTICALLY in every game:

1. MENU SYSTEM (same flow for all games):
   Tutorial → Main Menu → [Levels | Settings | Achievements | Stats | Daily] → Playing → Result

2. GAME HUD (same layout/buttons):
   - Timer (top center)
   - Score/Moves counter
   - Lives indicator (hearts)
   - Combo multiplier display
   - Buttons: Undo | Hint | Restart | Mute | Pause

3. SOUND SYSTEM:
   - Mute toggle on HUD (persists in localStorage)
   - Same key: '{gamename}_sound_enabled'
   - SFX categories: move, win, lose, combo, hint

4. SCORING (same formula structure):
   - Base points per action
   - Time bonus (faster = more points)
   - Combo multiplier (consecutive correct moves)
   - Star rating: ★ (complete) ★★ (under par moves) ★★★ (perfect + fast)

5. PROGRESSION:
   - 10 story levels per difficulty
   - Levels unlock sequentially (must complete previous)
   - Stars saved per level
   - localStorage persistence with same key patterns

6. LIVES SYSTEM:
   - Start with X lives per level
   - Lose life on wrong move / timeout
   - Game over when lives = 0
   - Can earn extra lives via combos or boosters

7. CELEBRATIONS:
   - Confetti on level complete
   - Star animation on perfect score
   - Screen shake on wrong move

8. SETTINGS SCREEN:
   - Sound on/off
   - Music on/off (when applicable)
   - Vibration on/off (mobile)
   - Reset progress button

9. STATISTICS:
   - Total games played
   - Total time played
   - Best scores per level
   - Win/loss ratio
   - Longest streak

10. DAILY CHALLENGE:
    - One unique puzzle per day (from backend seed)
    - Streak counter
    - Special reward for 7-day streak

SIMILAR GAMES REFERENCE (Flow Free)
============================================================

Flow Free by Big Duck Games:
  - 100M+ downloads on iOS/Android
  - Core mechanic: connect same-colored dots, fill entire grid
  - Grids: 5×5 to 15×15 (up to 16 colors)
  - Features: time trial mode, level packs, daily puzzle
  - "Perfect" game = fewest moves required (star reward)
  - Expansions: Bridges (pipes can cross), Hexes, Warps, Shapes
  - NP-complete problem (proven in 2022 paper)

Our "Block Fill" is essentially the same game with:
  - Premium dark neon aesthetic (vs Flow Free's flat colors)
  - Backend integration for tournaments/leaderboards
  - Phaser renderer option for performance
  - Level generator for infinite content
