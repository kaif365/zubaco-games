├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│ Indian law distinguishes between:                                               │
│ - Games of SKILL: Legal in most states (rummy, chess, fantasy sports)          │
│ - Games of CHANCE: Illegal gambling (slots, dice, lottery)                     │
│                                                                                 │
│ Supreme Court precedent (State of Andhra Pradesh v. K. Satyanarayana):         │
│ "If a game is PREDOMINANTLY skill-based, it is not gambling."                  │
│                                                                                 │
│ ZUBACO'S GAMES - Classification Analysis:                                      │
│                                                                                 │
│ CLEARLY SKILL-BASED (Safe):                                                    │
│   ✓ Sliding Puzzle - Pure logic, no randomness                                │
│   ✓ Maze Navigation - Spatial reasoning                                       │
│   ✓ Word Unscramble - Vocabulary skill                                        │
│   ✓ Speed Type Answer - Typing/knowledge skill                                │
│   ✓ Infinity Loop - Pure logic                                                │
│   ✓ Block Fill - Spatial reasoning                                            │
│                                                                                 │
│ PREDOMINANTLY SKILL (Likely safe):                                              │
│   ~ Sequence Recall - Memory skill with random sequences                      │
│   ~ Memory Card Matching - Memory skill with random card placement            │
│   ~ Flash Spot - Visual attention with random placement                       │
│   ~ Colour Sorting - Logic with random initial state                          │
│   ~ Arrows - Reaction time with random directions                             │
│   ~ Pattern Survival - Pattern recognition                                    │
│   ~ Logic Reflector - Logic puzzle                                            │
│                                                                                 │

┌─────────────────────────────────────────────────────────────────────────────────┐
│ 2.1 TDS ON ONLINE GAMING WINNINGS (Section 115BBJ)                              │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│ LAW: Finance Act 2023 introduced Section 115BBJ                                │
│ EFFECTIVE: April 1, 2023                                                       │
│ RATE: 30% TDS on NET WINNINGS                                                  │
│                                                                                 │
│ NET WINNINGS CALCULATION:                                                       │
│   Net Winnings = Withdrawals + Closing Balance - Deposits (in FY)             │
│                                                                                 │
│ TDS TIMING:                                                                     │
│   - At time of each withdrawal (if net winnings are positive)                 │
│   - At end of financial year (March 31) for remaining balance                 │
│                                                                                 │
│ EXAMPLE:                                                                        │
│   User deposits ₹10,000 in April 2026                                         │
│   Wins ₹5,000 in tournaments                                                  │
│   Balance: ₹12,000 (net of entry fees)                                        │
│   Requests withdrawal of ₹8,000                                               │
│                                                                                 │
│   Net winnings at withdrawal = ₹8,000 + 0 - ₹10,000 = -₹2,000               │
│   TDS = NIL (net winnings negative)                                            │
│                                                                                 │
│   Later wins more. Balance: ₹15,000                                           │
│   Requests withdrawal of ₹15,000                                              │
│   Cumulative: ₹23,000 withdrawn + ₹0 balance - ₹10,000 deposits = ₹13,000    │
│   Previous TDS: ₹0                                                            │
│   TDS on this withdrawal: ₹13,000 × 30% = ₹3,900                             │
│   User receives: ₹15,000 - ₹3,900 = ₹11,100                                  │
│                                                                                 │
│ PLATFORM OBLIGATIONS:                                                           │
│   ☐ Deduct TDS from each withdrawal/year-end                                  │
│   ☐ Deposit TDS with government within 30 days (challan 281)                  │
│   ☐ File 26QC quarterly (TDS return)                                          │
│   ☐ Issue Form 16C to each user (TDS certificate)                             │
│   ☐ Report on Form 61B if user doesn't have PAN (higher TDS 30% vs 30%)      │
│   ☐ PAN collection mandatory for all users with positive net winnings         │
│                                                                                 │
│ CURRENT IMPLEMENTATION: ████████████ ZERO                                      │
│   - No TDS calculation logic                                                  │
│   - No deposit tracking per financial year                                    │
│   - No PAN collection requirement                                             │
│   - No Form 16C generation                                                    │
│   - No quarterly filing system                                                │
│                                                                                 │
│ PENALTY FOR NON-COMPLIANCE:                                                     │
│   - Interest: 1.5% per month on unpaid TDS                                   │
│   - Penalty: Equal to TDS amount (Section 271C)                               │
│   - Prosecution: Up to 7 years imprisonment (Section 276B)                    │
│   - Disallowance of expense (Section 40(a)(i))                                │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
