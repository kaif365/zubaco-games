1. OVERVIEW
========================================================================
Reflex Endurance is a timed reflex game where colored circles spawn 
randomly on a play area. The player must tap ONLY the green circles 
while avoiding red, blue, and yellow circles. Circles disappear after 
2 seconds. Speed increases over time. Game ends on timer expiry or 
too many wrong taps.

Genre: Reflex / Reaction / Endurance
Core loop: Circle spawns → Player taps (or ignores) → Score/Penalty → Speed increases

4. GAME MECHANICS
========================================================================

4.1 PHASES
- idle: Shows InstructionScreen, player clicks "Start"
- playing: Circles spawn, player taps green ones
- ended: Shows ResultScreen with score

4.2 CIRCLE SYSTEM
- 4 colors: green (40%), red (20%), blue (20%), yellow (20%)
- Only GREEN is correct — all others are traps
- Position: random x (10-90%), random y (10-90%)
- Circles auto-disappear after 2 seconds
- Spawned by mulberry32(seed) for deterministic replay

4.3 SPEED ACCELERATION
- initialSpawnIntervalMs: 1200ms (default)
- speedIncreaseEveryMs: 30000ms (every 30s)
- speedMultiplier: 0.85 (speed * 0.85^N)
- Minimum interval: 300ms
- Formula: interval = max(300, initialInterval × multiplier^speedups)

4.4 GAME OVER CONDITIONS
- Timer runs out (timeLimitMs: 300000 = 5 minutes)
- Wrong taps reach maxWrongTaps (default: 2)

4.5 SCORING
- +1 point per correct green tap
- Wrong tap increments wrongTaps counter (no score deduction, just life loss)
- Endurance: survive as long as possible, tap as many greens as possible

4.6 CONTROLS
- Touch/Click on circles
- No keyboard controls for gameplay

4.7 DEFAULT CONFIG (GameConfig)
- timeLimitMs: 300000 (5 minutes)
- initialSpawnIntervalMs: 1200
- speedIncreaseEveryMs: 30000
- speedMultiplier: 0.85
- maxWrongTaps: 2

6. AUDIO (7 sounds)
========================================================================
- targetAppear: circle spawns
- targetHit: correct tap on green
- targetMiss: wrong tap on non-green
- combo: streak milestone
- levelComplete: game won / time survived
- gameOver: too many mistakes
- uiClick: menu interactions

========================================================================
7. BACKEND ENGINES
========================================================================
- achievementEngine: 10 achievements (First Win, Streak 5/10, Perfect, Speed Demon, etc.)
- boosterEngine: 5 boosters (Time Freeze, Hint, Skip, Double Points, Undo)
- comboEngine: streak tracking, multiplier
- dailyChallenge: SHA-256 date-based seed, difficulty by day of week, 1.5× bonus
- leaderboard: standard
- scoreHistory: standard
- levelConfig: 10 levels (above table)
- circleValidator: server-side tap validation
- timingAnalyzer: reaction time analysis

8. SAVE PROGRESS SYSTEM
========================================================================
- ReflexEnduranceProgressEntry: levelIndex, score, stars, completedAt, timeTakenMs, movesUsed, hintsUsed

- Functions: saveLevel, getProgress, getLevelProgress, getHighestCompletedLevel, getTotalScore
- Outbox pattern for offline sync
