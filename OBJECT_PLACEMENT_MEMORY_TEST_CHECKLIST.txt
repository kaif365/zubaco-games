NOTES
========================================================================
- Default config: 4×4 grid, 3 objects, 5s memorize, 90s recall
- Scoring: accuracy × 100 + floor(multiplier × remaining/total)
- Level progression: 4×4/3obj/5s → 7×7/12obj/1.8s
- Drag-and-drop: @dnd-kit/core (PointerSensor + TouchSensor)
- Object pool: 16 emojis across 5 categories
- Audio: 6 sounds (objectPlace, objectCorrect, objectWrong, etc.)
- State: Custom hooks, NOT Zustand
- Routing: react-router-dom (single route to GamePage)
- appPhase state machine wraps game phases in GamePage
- Auto-submit on timer expiry with useRef guard
