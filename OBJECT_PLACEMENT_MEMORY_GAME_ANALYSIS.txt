1. GAME CONCEPT
========================================================================
Object Placement Memory is a spatial recall game. Players are shown a
grid with objects (emojis) placed in specific cells for a limited time.
After the memorize phase, the grid clears and objects appear in a tray.
Players must drag-and-drop each object back to its remembered position.
Scoring rewards correct placements + speed.

Core loop:
  1. Grid displayed with objects in random cells (seeded)
  2. Memorize phase: player observes object positions (timed)
  3. Recall phase: grid empties, objects shown in tray (shuffled)
  4. Player drags objects from tray to grid cells
  5. Submit → score = accuracy × 100 + time bonus
  6. Results shown with correct/total breakdown

4. GAME MECHANICS DETAIL
========================================================================
Object Pool (16 items):
  Shapes: star, heart, diamond, moon, sun
  Nature: flower, tree, mushroom
  Food: apple, grape
  Animal: fish, bird, cat
  Misc: bolt, fire, snowflake

Board Generation:
  - Objects picked from pool via seeded shuffle
  - Cell positions picked from all grid cells via seeded shuffle
  - GridPlacement = { cellIndex, object }
  - Tray objects shuffled with offset seed (seed + 999)

Player Interaction:
  - DnD: PointerSensor + TouchSensor (mobile-friendly)
  - Drag object from tray → drop on grid cell
  - Click placed object to remove it back to tray
  - Object can be moved between cells (auto-removes from previous)
  - Submit button: shows count (placed/total)
  - Auto-submit when timer expires

Scoring:
  - accuracy = correctCount / totalObjects
  - baseScore = round(accuracy × 100)
  - timeBonus = floor(timeBonusMultiplier × remainingMs / recallTimeMs)
  - finalScore = baseScore + timeBonus
  - Default timeBonusMultiplier from config

Default Config (implied from backend L1):
  - gridSize: 4 (16 cells)
  - objectCount: 3
  - memorizeTimeMs: 5000 (5s)
  - recallTimeMs: 90000 (90s)
  - pointsPerCorrect: ~33 (100/total)
  - timeBonusMultiplier: 5
