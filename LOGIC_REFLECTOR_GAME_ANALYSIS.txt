───────────────────────────────────────────────────────────────────────────────
 1. GAME CONCEPT
───────────────────────────────────────────────────────────────────────────────

Logic Reflector is a laser-beam puzzle game. The player places mirror blocks,
splitters, and blockers on a grid to redirect one or more laser beams from 
emitters to hit all target nodes. The game is timed and scored — completing
levels faster and with fewer moves earns higher scores.

Core mechanic: Place reflective pieces (mirrors /, \, splitters +, blockers ■,
reflect-blocks □) on empty grid cells to guide the laser from emitters to
targets. The laser simulation runs in real-time, visually tracing beam paths
with bounces and splits.

Stack: React 19 + Vite + TypeScript + TailwindCSS + i18next + React Query
Entry: src/main.tsx → App.tsx → AppProviders → AppRouter → GamePage
Auth: Dev-session token auto-obtained on startup
Routing: Has router (GamePage, LevelEditorPage, NotFoundPage)

Components (src/features/game/components/):
  • GameContainer.tsx — Master orchestrator (themes, phases, auth, recovery)
  • GameBoard.tsx — Grid renderer (cells, placed blocks, laser beams, drag/drop)
  • GameCell.tsx — Individual cell (emitter icon, target icon, placed block)
  • LaserBeam.tsx — SVG beam path animation
  • Toolbar.tsx — Block palette (reflect-block, mirror-fwd, mirror-bwd, splitter, blocker)
  • HUD.tsx — Level counter + dots + quit button
  • DemoFlow.tsx — Tutorial/demo sequence with pre-built boards

Hooks (src/features/game/hooks/):
  • useGameSession.ts — Full game state machine (phases: idle→loading→level-opening→playing→level-clearing→loading-next→calculating_result→completed→error)
  • useLevelPrefetch.ts — Pre-fetches next level from API
  • useMoveSubmission.ts — Batched move submission to backend

Engine (src/game/):
  • laserEngine.ts — Ray-casting physics (reflect, split, block, exit detection)

Data (src/features/game/data/):
  • levels.ts — Local fallback levels (grid definitions)
  • demoData.ts — Demo boards for tutorial

Audio (src/audio/):
  • soundRegistry.ts — 7 sounds: mirrorPlace, beamHit, beamMiss, puzzleSolve, levelComplete, hint, uiClick
  • AudioManager.ts, AudioProvider.tsx, useAudio.ts

Micro-screens (src/micro-screens/):
  • GameInstructionsScreen.tsx — Pre-game instructions with stage theming
  • GameInstructionsSkeleton.tsx — Loading skeleton
  • GameResultScreen.tsx — Success/Failure result with score breakdown

Types (src/types/logic-reflector.ts):
  • Direction, CellType, BlockType, ServerCell, PlacedBlock, AvailableBlock
  • GameLevel, GameSession, GameMove, GameScoreboard, LevelScore
  • GameStatus enum (STARTED=1, ENDED=2, EXPIRED=3, RESULT_PROCESSING=4, MANUALLY_ENDED=5)
