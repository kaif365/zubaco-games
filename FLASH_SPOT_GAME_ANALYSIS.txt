GAME CONCEPT
============================================================

"Flash Spot" is a visual attention/reaction game. The player watches a grid
of coloured shapes and must TAP cells that momentarily change (color, shape,
size, or opacity). Changes appear briefly then revert.

RULES:
  1. A grid of cells (4×4 to 7×7) displays colored shapes
  2. Cells randomly change one property (color/shape/size/opacity) for a brief window
  3. Player must TAP the cell while it's changed → correct tap = points
  4. Tapping an unchanged cell → penalty
  5. Game ends when timer reaches 0
  6. Score = correct taps × points - wrong taps × penalty + time bonus

Difficulty scales by:
  - Grid size (4×4 easy → 7×7 hard)
  - Number of changes per round (3 easy → 10 hard)
  - Display duration (2000ms easy → 700ms hard)
  - Time limit (60s easy → 30s hard)
  - Score multiplier (1.0x → 3.5x)

SCORING SYSTEM
============================================================

Formula (client estimate):
  correctTaps × pointsPerCorrectTap (20)
  - wrongTaps × penaltyPerWrongTap (10)
  + timeBonus (floor(bonusTimeRatio × remainingSeconds))

With level multiplier (server): score × scoreMultiplier

Combo (backend):
  - Every consecutive correct tap increases streak
  - Multiplier = 1 + floor(streak/3) × 0.25 (max 4x)
  - Bonus accumulates: streak × 10 × multiplier
