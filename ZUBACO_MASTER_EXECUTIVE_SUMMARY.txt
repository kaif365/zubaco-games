WHAT ZUBACO IS:
A mass-participation, skill-based elimination gaming platform where
millions of players compete across 20 brain games in seasonal tournaments
for real-money prizes (Indian Rupees via Razorpay).

TECH STACK:
- Backend: NestJS 11 + Prisma 7 + PostgreSQL 16 + Redis 7 (Г—22 services)
- Admin: NestJS 11 + Prisma 7 (backend) + Next.js 16 + React 19 (frontend)
- Mobile: React Native 0.76 + React 18 + WebView
- Games: 20 Г— (React 19 + Vite 6 frontend + NestJS 11 + Prisma 7 backend)
- Infra: AWS (ECS Fargate, RDS, ElastiCache, ALB, CloudFront, WAF, SQS/SNS)
- IaC: Terraform >= 1.5
- CI/CD: GitHub Actions (3 workflows)
- Containers: Docker (multi-stage, alpine)

SCALE:
- 22 backend services (1 platform + 1 admin + 20 game backends)
- 21 frontend apps (1 admin + 20 game frontends)
- 1 mobile app (iOS + Android)
- 44+ package.json files
- ~750 lines of database schema (platform)
- ~600 lines of database schema (admin)
- ~100-200 lines per game schema (Г—20)
