1. GAME CONCEPT & GENRE
================================================================================

Genre: Relaxation Puzzle / Tile Rotation / Loop Connection
Core Mechanic: Tap tiles on a grid to rotate them 90°. Goal is to orient all
tiles so their connection lines form closed loops with no open ends.

Key Characteristics:
- No timer pressure (relaxation-focused)
- Single-tap interaction (rotate tile clockwise)
- Infinite procedural levels
- Minimalist aesthetic with calming visuals
- Closed-circuit connectivity validation

Tile Types (7 types used):
- EMPTY: No connections (blank space)
- CAP: One connection (dead end in a direction)
- STRAIGHT: Two opposite connections (horizontal or vertical pipe)
- ELBOW: Two adjacent connections (90° corner)
- TEE: Three connections (T-junction)
- CURVED_V: Like elbow but with curved rendering
- CROSS: Four connections (always correct regardless of rotation)

Win Condition: Every tile's connections must match perfectly with adjacent
tiles - no open ends, no mismatches between neighbors.

2. ONLINE RESEARCH — SIMILAR GAMES
================================================================================

─────────────────────────────────────────────────────────────────────────────────
A) "Infinity Loop: Relaxing Puzzle" by Infinity Games, Lda (Portugal)
   - 50M+ downloads, 4.6★, 813K reviews
   - THE original that defined this sub-genre
   - Features:
     • Infinite procedurally generated levels
     • No timer, no score, no pressure
     • Dark Mode (disconnect ALL pieces instead of connecting)
     • Lightbulb hint button (reveals one correct tile)
     • Minimalist flat design, dark background
     • Zen/ambient background music
     • Google Play Games leaderboard integration
     • Available on: Android, iOS, Windows (via Google Play Games), Wear OS
   - Player Feedback:
     • Too easy/monotonous after ~100 levels (patterns repeat)
     • Hint button reportedly unreliable
     • Ads too frequent (free version)
     • Users want progressive difficulty scaling
     • Users want level numbers/progress tracking
     • "Relaxing but lacks depth" after extended play

─────────────────────────────────────────────────────────────────────────────────
B) "Infinity Loop: HEX" by Infinity Games, Lda
   - 1M+ downloads, 4.3★, 117K reviews
   - Hexagonal grid variant of the same concept
   - Features:
     • Hex-based grid (6 directions instead of 4)
     • Same relaxation focus, no timer
     • Faint hex grid lines for visual clarity
     • More complex shapes due to 6-way connections
     • Anxiety relief / stress reduction branding
     • Wear OS support

     • More challenging than square grid version
     • Users request faint outlines to avoid misclicks

3. CURRENT CODEBASE ARCHITECTURE
================================================================================

Framework: Next.js (App Router) + TypeScript + Tailwind CSS + Framer Motion
State: Zustand store + React Context (GameContext)
Real-time: Socket.IO (WebSocket) for multiplayer/leaderboard
Audio: Howler.js (use-audio.ts = background, use-audio-effects.ts = SFX)
Rendering: Phaser.js (phaser-game-grid.tsx) + DOM fallback (game-grid.tsx)
i18n: react-i18next (multi-language support)
Backend: NestJS 11 + Prisma + Bun + Restate SDK

3.3 Game Flow (Current):
─────────────────────────────────────────────────────────────────────────────────

1. page.tsx (Tutorial) → GameProvider(TUTORIAL) → GameSection(instructions)
   → Shows InstructionsScreen (CMS carousel with stage overlays)
   → User clicks "Play Now"
   → Runs 2 tutorial rounds (TUTORIAL_LEVEL_COUNT = 2)

2. game/page.tsx (Live) → GameProvider(LIVE) → GameSection(live)
   → Connects WebSocket → Server sends board
   → User rotates tiles (tap = +90°)
   → Server validates rotations
   → Board solved → WinOverlay → Next board
   → Stage complete → GameClearModal → Redirect to result page

3. result/ → GameResultScreen (success/failure variant with stage overlay)

5. LEVEL CONFIGURATION
================================================================================

Level 1:  4x4 grid, STRAIGHT + ELBOW only, no time limit
Level 2:  4x4 grid, STRAIGHT + ELBOW + CAP, no time limit
Level 3:  5x5 grid, STRAIGHT + ELBOW + CAP, no time limit
Level 4:  5x5 grid, all tiles except CROSS, no time limit
Level 5:  5x5 grid, all tiles, 180s time limit
Level 6:  6x6 grid, all tiles, 180s time limit
Level 7:  6x6 grid, all tiles, 150s time limit
Level 8:  7x7 grid, all tiles, 150s time limit
Level 9:  7x7 grid, all tiles, 120s time limit
Level 10: 8x8 grid, all tiles, 120s time limit

Star Rating:
- 3 stars: Completed in ≤ optimal moves + 2
- 2 stars: Completed in ≤ optimal moves × 1.5
- 1 star: Completed (any number of moves)

Where optimal moves = number of non-EMPTY, non-CROSS tiles (minimum possible rotations).

================================================================================
6. DARK MODE / REVERSE MODE (UNIQUE FEATURE)
================================================================================

From the original Infinity Loop app:
- Normal Mode: All tiles scrambled → rotate to CONNECT all (form closed loops)
- Dark Mode: All tiles connected → rotate to DISCONNECT all (break all connections)

Implementation:
- In Dark Mode, win condition inverts: no tile should connect to any neighbor
- Same tiles, same grid, reversed goal
- Dark Mode uses inverted color scheme (bright tiles on dark = connected = BAD)
- Can be offered as alternate levels or unlockable mode after Level 5
