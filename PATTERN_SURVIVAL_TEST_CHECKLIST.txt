NOTES
========================================================================
- Survival mode: no win condition, just survive as long as possible
- Sequence grows indefinitely: round N = N+1 steps
- Wrong tap = immediate death (no lives system)
- Default: 3×3 grid, 6 colors, 5min timer, 500ms flash, 20pts/round
- Level progression: start length 3→8, display speed 800ms→300ms
- Audio: 8 sounds (patternShow, patternInput, patternCorrect, patternWrong, etc.)
- Backend: 5 boosters, combo engine (4× max), daily challenge
- State: Custom hooks (useState + useCallback + useRef), NOT Zustand
- No React Router — single GamePage with appPhase state machine
- Cell colors regenerate each round for visual variety
