NOTES
========================================================================
- Backend supports: 5 boosters (TIME_FREEZE, HINT, SKIP, DOUBLE_POINTS, UNDO)
- Backend has: achievement engine, combo engine, leaderboard (top 100), score history
- Combo: max 4× multiplier, milestone every 3, +0.25 per milestone, 10 bonus per level
- State management: useReducer (NOT Zustand) — all transitions via dispatch
- No React Router — single App.tsx with screen state machine
- Session encrypted with AES via VITE_ENCRYPTION_KEY
- Timer syncs with server (not pure client-side)
- Card flip animation: 400ms (FLIP_ANIMATION_DURATION_MS)
- Level complete celebration: 800ms delay
- Preview duration: varies per level (1-5 seconds)
- Mismatch display duration: from level config
- Daily challenge: SHA-256 seeded, level scales by day of week
- Star rating: 3★ = 0 mismatches, 2★ = ≤2, 1★ = completed
- Multiple card content types from API (symbol, image, color, wordImage)
